# PetCollector

This module is responsible for downloading the raw dataset of cat and dog images.

## Usage

1.  Install requirements:
    ```bash
    pip install -r requirements.txt
    ```
2.  Run the downloader:
    ```bash
    python main.py
    ```

The script will download 100 images of cats and 100 images of dogs into `../shared/data/raw/cat` and `../shared/data/raw/dog`.
